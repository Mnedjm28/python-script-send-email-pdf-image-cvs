# Python Email Sender with Attachments

## Description
This project is a **Python script** that allows you to send emails through **Gmail SMTP** with multiple types of attachments, including:

- PDF files  
- Excel/CSV files  
- Images  

The script demonstrates **sending emails programmatically**, handling attachments of different formats, and using **SSL/TLS for secure connections**.

---

## Features
- Send emails via Gmail SMTP server  
- Support for multiple attachment formats (PDF, Excel/CSV, images)  
- Secure connection using **STARTTLS**  
- Customizable email subject and content  
- Error handling to catch failed email attempts
